# Student Attendance Management System

A simple full-stack application for managing student attendance.

## Features

- Add students
- Mark attendance (present/absent)
- View attendance history
- Delete attendance records

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL

## Setup

1. Install Node.js and MySQL.
2. Create a MySQL database named 'attendance_db'.
3. Clone or download the project.
4. Navigate to backend directory: cd student-attendance/backend
5. Install dependencies: npm install
6. Start the server: npm start
7. Open frontend/index.html in a browser.

Note: Ensure MySQL is running and update db.js with your credentials if needed.
